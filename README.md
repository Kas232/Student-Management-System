# Student Management System

A simple and efficient **Student Management System** built using HTML, CSS, PHP and MYSQL.  
This project provides basic functionality to manage student records with authentication support.

## 🚀 Features
- User login and logout system
- Add new student records
- Edit existing student details
- Delete student records
- View all students in a structured table
- Responsive design with CSS styling

## 🛠️ Technologies Used
- PHP (Core logic and CRUD operations)
- MySQL (Database management)
- HTML & CSS (Frontend design)
- GitHub (Version control and collaboration)

## 📂 Project Structure
- `index.php` → Main entry point  
- `db.php` → Database connection  
- `login.php` / `logout.php` → Authentication system  
- `add_students.php` → Add new student  
- `edit_student.php` → Edit student details  
- `delete_student.php` → Delete student record  
- `view_students.php` → View all students  
- `style.css` → Styling and layout  

## ⚙️ Setup Instructions
1. Clone or download this repository.  
2. Import the provided SQL file (or manually create tables) into your MySQL database.  
3. Update `db.php` with your database credentials.  
4. Run the project on a local server (XAMPP/WAMP).  
5. Access via `http://localhost/Student-Management-System/index.php`.
